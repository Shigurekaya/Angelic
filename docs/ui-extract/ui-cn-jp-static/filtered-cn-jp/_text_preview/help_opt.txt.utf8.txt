# システム設定画面ヘルプテキスト

#------------------------------------------------------------------------------
# 共通

page0
	簡易設定タブへ移動します。
page1
	画像表示タブへ移動します。
page2
	ゲーム進行１タブへ移動します。
page3
	ゲーム進行２タブへ移動します。
page4
	テキストタブへ移動します。
page5
	サウンドタブへ移動します。
page6
	ダイアログタブへ移動します。
page7
	マウスタブへ移動します。
page8
	キーボードタブへ移動します。
page9
	ゲームパッドタブへ移動します。

back
	ゲーム画面に戻ります。
reset
	このページの設定を初期化します。	Shiftキーを押しながらボタンを選択することで、全てのページの設定を初期化します。
title
	タイトル画面に戻ります。

# 簡易設定

label_a_jump	label_b_jump
	詳細設定のために画像表示タブへ移動できます。

	詳細設定のためにテキストタブへ移動できます。

label_f_jump	label_g_jump	label_h_jump	label_i_jump	label_j_jump
	詳細設定のためにサウンドタブへ移動できます。

# 画面設定

fullscreen
	ウインドウモードとフルスクリーンモードの切り替えができます。

sqscr
	１６：９モードと４：３モードの切り替えができます。

noeffect
	画面効果のＯＮ・ＯＦＦを設定します。

scanim_on
	負荷の高いアニメーション効果のＯＮ・ＯＦＦを設定します。
scanim_off
	負荷の高いアニメーション効果のＯＮ・ＯＦＦを設定します。	ＯＦＦの場合は代替の静止画が使用されます。

esccancel_on
	ＥＳＣキーを押したときの機能を選択します。	右クリック：右クリック相当のキャンセル動作が割り当てられます。
esccancel_off
	ＥＳＣキーを押したときの機能を選択します。	スクランブル機能：下の欄で選択されたスクランブル機能が割り当てられます。

panictype_0
	スクランブル機能を選択します。	最小化：ゲームをタスクバーに収納して非表示にします。
panictype_1
	スクランブル機能を選択します。	画像１：ダミー画像１を表示します。同機能の再度実行で解除できます。
panictype_2
	スクランブル機能を選択します。	画像２：ダミー画像２を表示します。同機能の再度実行で解除できます。
panictype_3
	スクランブル機能を選択します。	ユーザー指定：指定の画像を表示します。決定後に画像ファイルを選んでください。

stayontop
	ゲーム画面を常に手前に表示する設定を変更します。

item_icon_chk
	ゲーム中の進行状態アイコンの表示を設定します。
item_vprg_chk
	ゲーム中のボイス再生やオート待ち時間の進行メーター表示を設定します。
item_qmlk_chk
	ゲーム中の画面下部の機能ボタンの表示を設定します。	ＯＦＦ状態でもマウスカーソルを当てると一時的に出現します。
item_toch_chk
	タッチパネル向けのユーザーインターフェース機能の表示を設定します。

chapthidetime
	ゲーム中のチャプター表示を設定します。左端で非表示、右端で常に表示、	中間は指定時間後（秒単位）に自動的に消去になります。

bgmhidetime
	ゲーム中に再生される音楽の曲名表示を設定します。左端で非表示、右端で常に表示、	中間は指定時間後（秒単位）に自動的に消去になります。


facemode_on
	ＯＮにすると立ち絵表示中も、テキストウインドウにキャラの表情を表示させます。	（常に表示させるわけではありません。演出上、表示されない箇所も存在します）
facemode_off
	ＯＦＦにすると立ち絵表示中は、テキストウインドウにキャラの表情を表示させません。	（常に非表示となるわけではありません。演出上、表示される箇所も存在します）

qcpopover
	ウインドウメニューに付随したポップアップ機能を表示する操作方法を設定します。

#------------------------------------------------------------------------------
# ゲーム進行

readskip
	既読テキストを自動的にスキップ/ジャンプするかどうかを設定します。	自動スキップ中は左クリックでスキップを一時停止できます。

readjump
	既読テキストの自動スキップ/ジャンプの動作を選択します。

curmove
	確認時のマウスカーソルの自動移動のＯＮ・ＯＦＦを設定します。

curmoveyes_on
	確認時のマウスカーソルの自動移動先を常に「はい」にします。
curmoveyes_off
	確認時のマウスカーソルの自動移動先を「いいえ」にします。	ただしクイックセーブロード等の一部の確認においては「はい」になります。


curhidestep_0
	マウスカーソルの自動消去を設定します。	マウスを停止した状態でもカーソルは常に表示されます。
curhidestep_5	curhidestep_10	curhidestep_20	curhidestep_x
	マウスカーソルの自動消去を設定します。	マウスを停止した状態で指定時間放置するとカーソル表示が消去されます。

filedclk_off
	セーブ/ロード操作をシングルクリックで実行します
filedclk_on
	セーブ/ロード操作をダブルクリックで実行します

hsel_area
	Ｈシーンの選択肢を固定するかどうかを設定します。

flowshow_on	flowshow_off
	未到達でもフローチャートの全体図を表示するかどうかの設定を変更します。

stopdeactive_on
	ゲームが非アクティブになった時の動作を設定します。	停止：ゲームの進行を一時停止して音声をミュートします。
stopdeactive_off
	ゲームが非アクティブになった時の動作を設定します。	継続：ゲームの進行を停止せずそのまま継続します。

icpreview
	タスクバーのプレビューの表示を設定します。
#	この機能はWindowsVista以降で有効です。

suspend_on	suspend_off
	ゲーム起動直後すぐに前回の続きを再開するかどうかの設定を変更します。

#===============================

drawspeed
	ゲームの再生速度を設定します。

vspeed
	音声の再生速度を設定します。	速度を上げると音質が劣化しますのでご注意ください。

vspeedsync_chk
	音声再生速度をゲーム再生速度と連動させます。	ただしx1.0～x3.0の範囲でのみ連動する動作となります。

skipspeed	skipspeed_rail
	スキップモード中の速度を設定します。	Ctrlキーによるスキップは、この設定に関係なく常に最速になります。

skipst_0
	スキップスタイルを設定します。	ノーマル：スキップ中でも全ての画面更新を表示します。
skipst_1
	スキップスタイルを設定します。	ファスト：スキップ中は立ち絵の表情やポーズ変更を省略します。
skipst_2
	スキップスタイルを設定します。	テキスト：スキップ中はテキストのみ表示し、画面や音楽は更新しません。

movierclick
	ムービー再生中に右クリックでスキップできるようにするかを設定します。	（右クリックのスキップはムービーが初視聴かどうかを問いません）

movieskip
	スキップ中に視聴済みのムービーに到達した際、スキップを継続するかを設定します。

drama_on
	ドラマチックモードのＯＮ・ＯＦＦを設定します。	ＯＮ：オートモード中、音声があるシーンはテキストウインドウを非表示にします。
drama_off
	ドラマチックモードのＯＮ・ＯＦＦを設定します。	ＯＦＦ：オートモード中、音声があるシーンでもテキストウインドウを表示します。

use_speech
	外部の音声ツールを連動し、ボイスがないテキストを読み上げます。	メニューバーの「ヘルプ／高度な設定／テキスト読み上げ」から詳細設定が可能です。

# テキスト設定

textspeed
	テキストの表示速度を設定します。

autospeed
	オートモードの基本待ち時間を設定します。${page==4?"\n基本待ち時間は、速度を実際の待ち時間に換算した値で、表記単位は秒です。":""}

atextwait
	オートモードの１文字当たりの追加待ち時間を設定します。	基本待ち時間にテキスト文字数分だけ追加して、最終的な待ち時間となります。


autovwait_0
	オートモード中のボイスの待ち方を設定します。	ノーマルモードは常にボイス再生を最後まで待ちます。
autovwait_1
	オートモード中のボイスの待ち方を設定します。ボイスモードは次の台詞に	ボイスがある場合にのみ最後まで待ち、ない場合は音声を継続して次に進みます。
autovwait_2
	オートモード中のボイスの待ち方を設定します。	スピードモードはボイスを待たずに常にオート速度の設定で進行します。

skipall
	既読テキストのみをスキップするか全てをスキップするかを設定します。

ctrlskip_on
	Ctrlキーを押した時のスキップ動作を設定します。	${_allSkip?"未読スキップが「すべて」の状態では設定できません。":"すべて：未読スキップの設定に関係なく強制で全てスキップします。"}
ctrlskip_off
	Ctrlキーを押した時のスキップ動作を設定します。	${_allSkip?"未読スキップが「すべて」の状態では設定できません。":"既読のみ：既読テキストのみをスキップします。"}

afterskip_on	afterskip_off
	選択肢を選んだ後もスキップを継続するかどうかを設定します。

afterauto_on	afterauto_off
	選択肢を選んだ後もオートモードを継続するかどうかを設定します。

color_win
	${getColorTagName("win")}の色を変更します。	このボタンを選択した後、右側の欄で色を設定してください。
color_owin
	${getColorTagName("owin")}の色を変更します。	このボタンを選択した後、右側の欄で色を設定してください。
color_hwin
	${getColorTagName("hwin")}の色を変更します。	このボタンを選択した後、右側の欄で色を設定してください。
color_text
	${getColorTagName("text")}の色を変更します。	このボタンを選択した後、右側の欄で色を設定してください。
color_read
	${getColorTagName("read")}の色を変更します。	このボタンを選択した後、右側の欄で色を設定してください。

nohwindow_chk
	Ｈシーン時に小サイズのウインドウ表示を使用するかを設定します。	チェックを入れて無効にしても、ウインドウ色の変更は動作します。

hsv
	現在の${getColorTagName()}の色の状態です。	${hsv?"丸いピンをドラッグして色を決定してください。":"クリックすると色設定ダイアログが開きます。"}

hsv_reset
	${getColorTagName()}の色を初期設定に戻します。

HSV_DRAG_H
	${getColorTagName()}の明度・彩度を変更中です。
HSV_DRAG_SV
	${getColorTagName()}の色相を変更中です。

winsample
	現在の各種設定のサンプルが表示されています。

winopac
	ゲーム中のテキストウインドウの透明度を設定します。

fontselect
	フォントを変更します。
