# タッチＵＩカスタマイズ画面ヘルプテキスト

FIRST
	ドラッグ＆ドロップで${!qmEditMode?"タッチパネル用ＵＩ":"ウインドウメニュー"}のボタン配置を変更できます。

DRAG
	ドラッグして上のバーの配置したい場所にドロップしてください。

DROP
	配置したい場所にドロップしてください。同じ種類のボタンは複数配置できません。

ONDRAG1
	${DROP}	（操作によっては別の場所の同じボタンが消える場合があります）

ONDRAG2
	${DROP}	（このボタンはバーから外すことはできませんのでご注意ください）


preview
	実際のゲーム中での表示のプレビューです。サイズや透明度を確認できます。	ここにはドラッグでボタンを配置できません。配置は下にあるバーを使用してください。

icons_area
	現在のボタン配置状況です。左右にドラッグして順番を入れ替えることができます。	上の何もないところにドロップすると配置から消すことができます。

tab_window
	ウインドウメニューのボタンカスタマイズを行います。
tab_touch
	タッチＵＩメニューのボタンカスタマイズを行います。
menulink
	ウインドウメニューとタッチＵＩメニューの配置を統一します。
entouch	qmlock
	${!qmEditMode?"タッチパネル用ＵＩ":"ウインドウメニュー"}の表示を有効にします。

func0

ITEM_SAVE
	セーブ画面を開くボタンです。

ITEM_LOAD
	ロード画面を開くボタンです。

ITEM_QSAVE
	クイックセーブするボタンです。

ITEM_DSAVE
	ダイレクトセーブするボタンです。

ITEM_QLOAD
	クイックロードするボタンです。

ITEM_OPTION
	システム設定画面を開くボタンです。

ITEM_VREPLAY
	現在の音声をリピート再生するボタンです。

ITEM_VSAVE
	お気に入りボイス画面を開くボタンです。

ITEM_QVSAVE
	お気に入りボイス登録するボタンです。

ITEM_PREVSCN
	前のシーンへジャンプするボタンです。

ITEM_PREV
	前の選択肢へジャンプするボタンです。

ITEM_BACKONE
	１つ前のテキストに戻るボタンです。

ITEM_BACKSKIP
	バックスキップモードを切り替えるボタンです。

ITEM_LOG
	バックログ画面を開くボタンです。

ITEM_AUTO
	オートモードを切り替えるボタンです。

ITEM_SKIP
	スキップモードを切り替えるボタンです。

	次のシーンへジャンプするボタンです。

ITEM_NEXT

ITEM_SCNCHART
	フローチャート画面を開くボタンです。

ITEM_VOLUME
	音量を変更するボタンです。

ITEM_HIDE
	テキストウインドウを一時的に非表示にするボタンです。

ITEM_TITLE
	タイトル画面に戻るボタンです。

ITEM_SNAPSHOT
	スクリーンショットを保存するボタンです。

sl_size
	タッチパネル用ＵＩの大きさを変更できます。	ゲーム中にピンチイン・ピンチアウトのタッチ操作でいつでも大きさを変えられます。

useqvsave
	お気に入りボイス登録のボタンの機能を切り替えます。

usedsave
	SAVEボタンの機能を切り替えます。
	


sl_opac
	タッチパネル用ＵＩの透明度を変更できます。

init
	${_helpTabName}のカスタマイズの内容を初期状態に戻します。
NAME_WINDOW
	ウインドウメニュー
NAME_TOUCH
	タッチＵＩメニュー


revert
	変更した各種設定をこの画面が開いた時の状態まで戻します。	※実行確認表示は無いので注意してください。

ok
	ボタンカスタマイズ画面を閉じます。
