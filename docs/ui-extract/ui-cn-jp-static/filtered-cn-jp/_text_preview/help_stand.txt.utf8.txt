# 立ち絵鑑賞画面ヘルプテキスト

back	title	title2
	タイトル画面に戻ります。

to_cg
	ＣＧ鑑賞画面に移動します。
to_scene
	シーン鑑賞画面に移動します。
to_voice
	お気に入りボイス画面へ移動します。
to_stand
	立ち絵鑑賞画面に移動します。

?meswin
	テキストウインドウを表示し、任意のテキストを表示できます。	（テキストは画像保存時や操作ボタン非表示時に見ることができます）

edit_save
	現在の状態をファイルに保存します。	Shiftキーを押しながら選択するとメモリに一時保存します。

edit_load
	保存されたファイルから状態を復帰させます。	Shiftキーを押しながら選択するとメモリに一時保存されたデータから復帰させます。

edit_hide
	操作ボタンを一時的に非表示にして、ドラッグ操作でキャラクターや背景（固定壁紙を除く）の位置を変更できるようにします。（右クリックと同等の機能です）

edit_capt
	操作ボタンを一時的に非表示にして、スクリーンショット画像を保存します。

edit_mybg
	背景画像にユーザー指定の画像を読み込んで表示します。

edit_init
	選択中のキャラクターを初期状態に戻します。（位置もリセットされます）	Shiftキーを押しながら選択すると現在の全ての状態を破棄して最初の状態に戻します。


dress_frame	HELP_CHSEL_DRESS
	現在のキャラクターの服装を変更できます。
pose_frame	HELP_CHSEL_DIFF
	現在のキャラクターのポーズを変更できます。
face_help	HELP_CHSEL_FACE
	現在のキャラクターの表情を変更できます。

fopt_h	fopt_n	fopt_r	fopt_d	fopt_e
	表情のバリエーションを変更できます。	選択中のボタンを再度クリックすることで標準状態に戻ります。

chzoom_frame
	現在のキャラクターの大きさを変更できます。	（他のキャラクターとの前後関係は維持されます）

stage_frame	HELP_BGSEL_STAGE
	背景を変更できます。
time_frame	HELP_BGSEL_TIME
	背景の時間帯／特殊差分を変更できます。
?chsel_pageup	?chsel_pagedown	?bgsel_pageup	?bgsel_pagedown
	サムネイル一覧をスクロールします。
?chsel_close	?bgsel_close
	サムネイル一覧表示を閉じます。	右クリックでも閉じることができます。

chlock
	操作対象のキャラクターの位置と大きさを固定して動かせないようにします。	また、消去ボタンも無効にします。

?chadd
	操作対象のキャラクターを追加します。
?chpale
	操作対象のキャラクター以外を半透明で薄く表示します。ONにすると、ドラッグの移動対象が現在の操作対象キャラクターで固定になります。

chdel
	現在操作中のキャラクターを消去します。${!_chlocked?"":"\n※このキャラクターを消去するには、ロックを解除してください。"}
?chfront
	操作対象のキャラクターを変更した際に、手前に表示するかどうかを設定します。
?chsort
	キャラクター一覧を、大きいものを手前の順に並べ替えます。


HELP_DRAG
	キャラクターをクリックで選択、ドラッグで移動することができます。
HELP_NODRAG
	このキャラクターはドラッグで移動することができません。	移動するにはクリックで選択後、ロックを解除してください。

HELP_CHIDX
	操作キャラクターを変更します。
HELP_CHIDX_DRAG
	操作キャラクターを変更します。	ドラッグして表示優先順位を変更できます。

HELP_CHDEL
	このキャラクターを消去します。

HELP_CHDRAG
	優先順位を変更したい位置までドラッグしてください。
